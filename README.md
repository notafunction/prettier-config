# @notafunction/prettier-config

Personal prettier configuration

## Usage

### Install:

```bash
$ npm i --dev @notafunction/prettier-config
```

### Add `.prettierrc`
```jsonc
"@notafunction/prettier-config"
```

### Or in `package.json`:

```jsonc
{
  // ...
  "prettier": "@notafunction/prettier-config"
}
```
